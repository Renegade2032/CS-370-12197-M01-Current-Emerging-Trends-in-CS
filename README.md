# CS-370-12197-M01-Current-Emerging-Trends-in-CS

Project Overview
For this project, I worked on creating a pirate intelligent agent using reinforcement learning. The main idea was to get the agent to learn how to make better decisions over time by using rewards and penalties. At first, it didn’t really perform well, but as training went on and I adjusted things, it started improving and reaching the goal more consistently.
What code was provided vs. what I created
Some of the setup was already given, like the environment and parts of the training process. My part was filling in the logic for how the agent actually learns, including how it updates its decisions based on rewards. I also had to mess with different values and fix issues when things weren’t working right. A lot of it was trial and error to get the agent to behave the way it was supposed to.

Reflection Questions
What do computer scientists do and why does it matter?
Computer scientists basically solve problems using technology. That could be writing code, building systems, or figuring out better ways to handle data. It matters because so much of what we use every day depends on it, even stuff people don’t really think about like recommendation systems or automation behind the scenes.
How do I approach a problem as a computer scientist?
I usually start by trying to understand what the problem is actually asking, then break it down into smaller pieces. From there I try something, test it, and fix it when it doesn’t work (which happens a lot). This project really showed me that especially with machine learning, you don’t just get it right the first time—you have to keep adjusting things until it starts working.
What are my ethical responsibilities?
I think one of the biggest responsibilities is making sure what I build doesn’t negatively affect people. That means thinking about things like fairness, security, and how the system will actually be used. It’s also important to be honest about what the system can and can’t do, and not ignore potential problems just to get something working.
